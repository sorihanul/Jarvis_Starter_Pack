# 1. 목적 (Purpose)
특정 키워드나 주제에 대해 외부 지식 및 로컬 문서를 검색(`IVK`)하여 가장 핵심적인 팩트와 출처를 요약합니다.

# 2. 입력 (Input)
- `목표 키워드 혹은 질의문`
- (선택사항) `허용된 검색 경로` (예: "04_Knowledge 폴더 한정")

# 3. 절차 (Procedure)
1. 요청받은 키워드의 우선적인 위치를 `04_Knowledge` 또는 웹 환경에서 탐색합니다.
2. 관련된 문서나 URL의 본문을 읽고, 질문에 부합하는 핵심 정보 블록을 추출합니다.
3. 관련 없는 군더더기 서술을 제거하고 팩트 위주로 압축합니다.
4. 문장마다 출처(파일 경로 혹은 URL)를 주석이나 괄호 형태로 표기합니다.

# 4. 출력 (Output)
- **헤더**: [요약 리포트: 주제명]
- **본문**: 3단락 이내의 핵심 팩트 서술
- **참고 출처**: 리스트 형태로 사용된 원본 파일명 혹은 링크 명시

# 5. 실패 처리 (Fallback)
- **정보 없음**: 검색 결과가 없거나 관련성이 떨어지면, 억지로 생략된 정보를 만들어내지 않고 "수집된 정보가 부족합니다. 키워드를 변경하시겠습니까?"라고 응답하며 종료합니다.
