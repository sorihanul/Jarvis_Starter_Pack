# 1. 목적 (Purpose)
현재 수행된 턴(Turn)이나 세션(Session)의 대화 기록과 작업 성과를 짧고 강력한 캡슐화 포맷으로 압축하여 보관합니다. 다른 에이전트나 미래의 세션에서 이를 인수인계 메모로 활용할 수 있도록 돕습니다.

# 2. 입력 (Input)
- `현재 세션의 맥락 전체` 혹은 `가장 최근에 완료된 Task의 히스토리`

# 3. 절차 (Procedure)
1. 현재까지 진행된 대화, 수행한 명령어, 생성된 파일 내역을 쭉 스캔합니다.
2. 인사말, 반복된 오류, 실패했던 시도 등 불필요한 '과정'을 쳐냅니다.
3. "무엇을 결론지었는가", "어떤 파일을 최종 수정했는가"의 '결과'만 남깁니다.
4. 내용을 `CAPSULES/` 디렉토리에 저장할 수 있는 표준 마크다운 형식으로 렌더링합니다.

# 4. 출력 (Output)
- **생성될 파일명**: `CAPSULES/Session_YYYYMMDD_HHMM.md` 형식
- **내용물**:
  - `작업 요약`: 1~2줄 핵심 내역
  - `수정/생성 파일`: [경로/파일명] 목록
  - `후속 조치 (Todo)`: 아직 남은 과제나 전달 사항

# 5. 실패 처리 (Fallback)
- **요약할 내용 부재**: 아무런 작업도 수행되지 않은 상태에서 본 스킬이 호출되면 "진행된 작업 내역이 없어 캡슐을 생성하지 않습니다."를 반환합니다.
