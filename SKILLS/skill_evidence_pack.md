# 1. 목적 (Purpose)
리서치 결과를 단순 요약이 아니라, 근거와 불확실성을 가진 bounded evidence pack으로 정리합니다.

# 2. 입력 (Input)
- `질문 또는 조사 주제`
- `수집된 메모, 파일, 경로, 링크`

# 3. 절차 (Procedure)
1. 핵심 주장 1~3개를 추립니다.
2. 각 주장에 대응하는 근거 파일 또는 출처 경로를 붙입니다.
3. 확실하지 않은 부분은 따로 `uncertainty`로 분리합니다.
4. 다음 단계에 필요한 추가 조사 또는 검증 포인트를 한 줄로 남깁니다.

# 4. 출력 (Output)
- **claims**: [핵심 주장 1~3개]
- **evidence_paths**: [근거 경로 또는 출처]
- **uncertainty**: [남은 불확실성]
- **next_probe_if_needed**: [추가 확인 지점]

# 5. 실패 처리 (Fallback)
- **근거 빈약**: 근거가 약하면 단정하지 말고 `uncertainty`를 키우며 종료합니다.
