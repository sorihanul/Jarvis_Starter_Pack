# Madang-Pan Lite Starter

이 모듈은 GPTs용 `Madang-Pan` 원형을 `Jarvis Starter`에 맞게 경량화한 전면 라우터입니다.

목적:
- 사용자 턴을 먼저 분류
- 짧은 브리핑을 먼저 표시
- 한 번에 하나의 현재 단계만 유지
- 스타터 팩에서 과도한 멀티에이전트 연극을 막음

이 모듈이 하는 일:
1. 요청을 `기본응답 / 분석 / 설계 / 검토 / 제작 / 번역` 중 하나로 분류
2. `진입 -> 정의 -> 작업 -> 검토 -> 출력`의 얕은 단계 흐름 제공
3. 스타터 팩에서 바로 쓸 수 있는 짧은 출력 계약 제공

이 모듈이 하지 않는 일:
- 실제 프로젝트 런타임 파일 생성
- Codex App식 프로젝트 루트 운영
- 외부 연합 레인 관리

읽기 순서:
1. `00_MADANG_PAN_LITE_ORCHESTRATOR_v0.1.md`
2. `10_MADANG_PAN_LITE_REGISTRY_v0.1.md`
3. `20_MADANG_PAN_LITE_BRIEFING_AND_OUTPUTS_v0.1.md`

권장 사용 위치:
- 스타터 팩 초기 대화 진입부
- 초보 사용자에게 작업 흐름을 더 선명하게 보여줘야 할 때
- 하나의 세션에서 순차 빙의형으로 응답할 때
