# Session Close Protocol v0.1

## 목적

세션이 끝날 때마다 최소한의 학습 흔적을 남겨 다음 작업에 재사용 가능하게 만든다.

## 종료 순서

1. 결과 요약
- 이번 세션에서 실제로 끝난 것

2. 실패/막힘 요약
- 다시 막힐 가능성이 큰 지점

3. 재사용 포인트 추출
- 반복 가능한 절차
- 반복 가능한 역할
- 반복 가능한 체크리스트

4. 캡슐 기록
- `CAPSULES/CAPSULE_TEMPLATE.md` 기준으로 1장 작성

5. 승격 후보 기록
- `TASKS/PROMOTION_CANDIDATE_TEMPLATE.md` 기준으로 필요시 작성

## 한 줄 원칙

전체 로그를 보존하려 하지 말고, 다음 세션에 다시 쓸 수 있는 최소 교훈만 남긴다.
