# Learning Loop Lite Starter

이 모듈은 스타터 팩에 `가벼운 학습 루프`만 추가한다.

목적:
- 세션 종료 시 핵심 교훈을 남긴다.
- 스킬/에이전트 승격 후보를 분리해서 기록한다.
- 자동 승격 없이도 자산화를 시작할 수 있게 한다.

핵심 구성:
1. `SESSION_CLOSE_PROTOCOL_v0.1.md`
2. `PROMOTION_CHECKLIST_v0.1.md`
3. `CAPSULES/CAPSULE_TEMPLATE.md`
4. `TASKS/PROMOTION_CANDIDATE_TEMPLATE.md`
5. `OPENVIKING_SCOPE_NOTE_v0.1.md`

원칙:
- 학습 후보는 남기되, 자동 승격은 하지 않는다.
- 긴 로그 대신 압축된 캡슐만 남긴다.
- 반복 검증된 것만 `SKILLS` 또는 `AGENTS` 승격 후보가 된다.
- `OpenViking`은 전면 도입하지 않고, 계층형 요약과 세션 커밋 사고방식만 참고한다.
